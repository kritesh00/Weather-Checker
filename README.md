# Weather Checker

A React + Vite application that fetches and displays weather information for Kathmandu using the GoWeather API. Built with modern web technologies including React 19, Vite 7, and Tailwind CSS 4.

## Overview

This project uses the Fetch API to retrieve real-time weather data from the GoWeather service. It features a beautiful, responsive UI with snowfall effects powered by react-snowfall.

## API Endpoint

**Base URL:** `https://goweather.xyz/weather/{city}`

The API returns current weather conditions including temperature, wind speed, and weather description for differnt cities.

## Tech Stack

- **React 19.2** - Modern React with latest features
- **Vite 7.3** - Lightning-fast build tool and dev server
- **Tailwind CSS 4.1** - Utility-first CSS framework
- **react-snowfall** - Beautiful snowfall animation effects
- **ESLint** - Code quality and consistency
